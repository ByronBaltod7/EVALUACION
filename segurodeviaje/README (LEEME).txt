*Se utilizó el Framework CodeIgniter 3.1.11 bajo el lenguaje PHP

*Con la versión Framework CodeIgniter 3.1.11

*Pasos para instalar en una máquina local 
1° Instalar Xampp V3.2.2
2° Descargar el código del repositorio GitHub
3° Entrar a la ubicación donde se instaló el Xampp e ingresar a la carpeta htdocs
4° Crear la carpeta EVALUACIÓN y dentro de ella guardar el proyecto.
